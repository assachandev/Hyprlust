# ✨ HYPRLUST

A Hyprland desktop configuration for Arch Linux — featuring dynamic wallpaper theming, 23 Waybar layouts, and a fully integrated Wayland workflow.

> **Forked from [NischalDawadi/Hyprlust](https://github.com/NischalDawadi/Hyprlust)**
> Original configuration by [NischalDawadi](https://github.com/NischalDawadi). This fork adds an Arch Linux installer and personal config adjustments.

## ⚡ Quick Install (Arch Linux)

```bash
git clone https://github.com/assachandev/Hyprlust.git ~/Hyprlust
cd ~/Hyprlust
bash install.sh
```

> Requires a fresh Arch Linux install with a display manager already set up.
> The script will install all packages, copy configs, and enable services automatically.

**After installation:**
1. Put wallpapers in `~/Pictures/wallpapers/`
2. Log out and select **Hyprland** from your display manager
3. On first boot run: `hyprpm reload`

## Core Stack

| Category | Tool |
|----------|------|
| Window Manager | [Hyprland](https://hyprland.org/) |
| Status Bar | [Waybar](https://github.com/Alexays/Waybar) — 23 layout presets |
| App Launcher | [Rofi](https://github.com/davatorium/rofi) |
| Terminal | [Kitty](https://sw.kovidgoyal.net/kitty/) |
| Notifications | [SwayNC](https://github.com/ErikReider/SwayNotificationCenter) |
| Wallpaper | [awww](https://github.com/aditya-K2/awww) + wallust auto-theming |
| Lock Screen | [Hyprlock](https://github.com/hyprwm/hyprlock) |
| Idle Daemon | [Hypridle](https://github.com/hyprwm/hypridle) |
| File Manager | [Yazi](https://github.com/sxyazi/yazi) |
| Audio | PipeWire + Wireplumber |
